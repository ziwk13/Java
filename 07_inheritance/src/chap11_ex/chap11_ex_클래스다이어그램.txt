클래스 다이어그램 (상속 관계)

+---------------------------+
|       <<interface>>       |
|      chap11_ex.Shape      |  // 도형
+---------------------------+
|                           |
+---------------------------+
| + getArea() : double      |  // 넓이 반환
+-------------+-------------+
              ▲
              |
+-------------+-------------+
|    chap11_ex.Rectangle    |  // 직사각형
+---------------------------+
| - width : double          |  // 너비
| - height : double         |  // 높이
+---------------------------+
| + Rectangle()             |
| + Rectangle(width:double, |
|   height:double)          |
| + getArea() : double      |
+-------------+-------------+
              ▲
              |
+-------------+-------------+
|    chap11_ex.Square       |  // 정사각형
+---------------------------+
|                           |
+---------------------------+
| + Square()                |
| + Square(side:double)     |  // 정사각형의 한 변의 정보 전달
+---------------------------+

* 필요한 경우 각 클래스의 Getter/Setter 메소드는 직접 구현합니다.
